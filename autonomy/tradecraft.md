# Tradecraft — Search Heuristics

## By Information Type

| I need... | Best source | Query that works | What does NOT work |
|-----------|------------|------------------|--------------------|
| Recent paper with code | Papers With Code > ArXiv | search by task name | searching by author |
| Real production experience | X (practitioners) > HN comments | topic search from builders | generic queries |
| Deep technical debate | Reddit r/MachineLearning > HN | subreddit search, sort by top | front page (too broad) |
| State of the art by task | Semantic Scholar | query with task name, sort by citations | overly specific queries |
| Emerging tool | HN Show > GitHub trending | topic with min-points filter | searching by name (don't know the name yet) |
| Official documentation | WebFetch direct URL | site:docs.example.com | generic WebSearch |
| Comparative benchmarks | Papers With Code > HN | benchmark name + "results" | generic Google |
